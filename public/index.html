Task Manager API

Built with Node.js + Express to manage your tasks

API is Running

Endpoints

GET /api/tasks

Get all tasks from the database.

POST /api/tasks

Create a new task (title and description required).

PUT /api/tasks/:id

Update a task’s completion status or content.

DELETE /api/tasks/:id

Remove a task permanently by ID.
